# Maths-The-Bomb
Online math web game - Answer quick and don't blow up

Playable at https://maths-the-bomb.onrender.com (if down may take a few seconds to come back up)

# Description 
Made as a educational math game project it has players passing around a bomb with a timer needing the player to answer math question quickly to pass it on. Targeted toward elementary-middle but is also good at working on fast calculations.

# Math Topics
- Arithmetic(addition,subtraction,multiplication,division)
- PEMDAS
- Shapes
- Clock reading

# Features
- Quick Match (Play online games against other players)
- Private Lobbies (Create your own lobbies to play with a specific group)
- Play with bots (Train your skills in a game against robots)
- Singleplayer (play by yourself and see how far you can go)
- Customization (Customize your self with different cosmetics)
- Difficulty selector
